# Voltrite

**Voltrite C** is a `C` project templet for a simple `C` projects. 

---

## USAGE

Clone the repository:
```bash
git clone https://gitea.com/oarix/Voltrite.git
```

>  - Change the [project.mk](project.mk) file to get started.  
> - Main file is <project_name.c>
